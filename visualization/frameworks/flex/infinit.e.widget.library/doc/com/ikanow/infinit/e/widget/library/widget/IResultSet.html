getGeoCounts()method 
public function getGeoCounts():ArrayCollection

If available (either directly from the query, or from the documents), an array collection of geotags (format: {lat:number, lon:number, count:number}) ordered by lat/long. Note that due to the granularity of the geohash used to aggregate lat/longs, distinct elements can have the same lat/long - since they will always be adjacent in the array, they can simply be summed.

Returns
