filterByDocField()method 
public function filterByDocField(filterDataSet:FilterDataSetEnum, values:Set, field:String = _id, filterDescription:String = null):void

Sends a filter event that will be sent to all widgets (including this one) This filter event takes a field and a list of items to filter on. This call will result in a widgets onReceiveNewFilter() function to be called.

Parameters

filterDataSet:FilterDataSetEnum — The dataset that needs filtered, either the global query data set of the filtered dataset is being further filtered
 
values:Set — The set of string values to search a documents field for e.g. if field is _id then values would be a list of document _id numbers to filter on
 
field:String (default = _id) — The field to filter a document on, defaults to _id. This can be any field that a document contains, for example title, source, etc
 
filterDescription:String (default = null) — A string that will be displayed on the tooltip of the breadcrumb for this filter. If left null a default one will be made for this filter.

Sends a filter event that will be sent to all widgets (including this one) This filter event takes a field and a list of items to filter on. This call will result in a widgets onReceiveNewFilter() function to be called.

docToEntityField:String (default = _id) — The field that docToEntMap is mapping on, defaults to _id

filterByEntities()method 
public function filterByEntities(filterDataSet:FilterDataSetEnum, entDisNames:Set, matchType:EntityMatchTypeEnum, includeEntsType:IncludeEntitiesEnum, filterDescription:String = null):void

Sends a filter event that will be sent to all widgets (including this one) This filter event takes a set of entity names and filters documents on those names using the specified params This call will result in a widgets onReceiveNewFilter() function to be called.

Parameters

filterDataSet:FilterDataSetEnum — The dataset that needs filtered, either the global query data set of the filtered dataset is being further filtered
 
entDisNames:Set — A set of entity names to filter on
 
matchType:EntityMatchTypeEnum — How the data should be filtered: ALL means that a document must contain every entity in entDisNames while ANY means that a document must contain atleast 1 entity in entDisNames
 
includeEntsType:IncludeEntitiesEnum — Determines if all the entities of a matching document should be added to the filter dataset or just the entities that were matched on: INCLUDE_ALL_ENTITIES means take all a docs entities while INCLUDE_SELECTED_ENTITIES only takes matching entities (i.e. will be a subset of entDisNames)
 
filterDescription:String (default = null) — A string that will be displayed on the tooltip of the breadcrumb for this filter. If left null a default one will be made for this filter.
